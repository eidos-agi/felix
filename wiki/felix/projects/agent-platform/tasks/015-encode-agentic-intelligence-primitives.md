# Encode Agentic Intelligence Primitives

## Goal

Felix-built agents should inherit "The Primitives of Agentic Intelligence" as doctrine, not just fetch the gist as context.

## Result

Felix standards now name thinking, tools, memory, coordination, and goal orientation as required agent architecture primitives.

The standards also encode the key asymmetries: memory is thinking substrate, tools are instruments for world access rather than authorities, coordination is a meta-layer, intelligence is hierarchical, and self-improvement organizes around a North Star.
